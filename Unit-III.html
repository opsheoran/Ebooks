3.1 What is a Random Variable?

Before we start with the formal definition, let us think about it with a simple everyday situation.

Suppose you toss a coin twice. You know that the possible outcomes are: HH, HT, TH, TT. Now if someone asks you — "How many heads did you get?" — your answer will be 0, 1, or 2, depending on what happened. This answer changes every time you do the experiment. It is numerical, and it depends on chance. This is exactly what a random variable is.

3.1.1 Definition

Definition 3.1.1 — Random Variable

A random variable (usually denoted by capital letters \(X\), \(Y\), \(Z\)) is a real-valued function defined on the sample space \(S\) of a random experiment. It assigns a numerical value to each outcome of the experiment.

Mathematically, we write:

\[X:S\rightarrow \mathbb{R}\]

That is, \(X\) maps every outcome \(\omega \in S\) to a real number \(X(\omega)\).

In plain words: A random variable converts the outcomes of a random experiment into numbers so that we can do mathematics on them.

3.1.2 Why Do We Need Random Variables?

In real experiments, outcomes are often non-numerical — "Head", "Tail", "Red ball", "Defective item", and so on. Mathematics does not work well with words. A random variable converts these wordy outcomes into numbers, making it possible to apply algebraic and calculus tools.

3.1.3 Examples

Example 3.1.1 — Tossing a coin twice

Experiment: Toss a fair coin twice.

Sample space: \(S = \{ HH,\ HT,\ TH,\ TT\}\)

Define \(X\) = number of heads obtained.

OutcomeValue of \(X\)

So \(X\) takes values \(0,1,2\). This is a random variable.

Example 3.1.2 — Rolling a die

Experiment: Roll a fair six-sided die once.

Sample space: \(S = \{ 1,2,3,4,5,6\}\)

Define \(X\) = number on the top face. Then \(X \in \{ 1,2,3,4,5,6\}\). Since every outcome is already a number, the mapping is straightforward.

Example 3.1.3 — Drawing a card

Experiment: Draw one card from a standard deck of 52 cards.

Define \(X\) = number on the card (Ace = 1, Jack = 11, Queen = 12, King = 13).

Then \(X\) takes values \(1,2,3,\ldots,13\).

Example 3.1.4 — Measuring rainfall

Experiment: Record the rainfall in your city on a particular day.

Define \(X\) = amount of rainfall in mm.

\(X\) can take any value in \(\lbrack 0,\infty)\). This is also a random variable — we will see later that it is a continuous random variable.

Example 3.1.5 — Counting defective items

A batch of 50 bulbs is inspected. Define \(X\) = number of defective bulbs found.

\(X\) can take values \(0,1,2,\ldots,50\).

Important Note: A random variable is not really a "variable" in the usual algebra sense. It is a function. The word "random" indicates that its value is determined by the outcome of a random (chance-based) experiment.

3.2 Discrete and Continuous Random Variables

Random variables are classified into two main types based on the nature of values they can take. This classification is very important because the methods we use for them are completely different.

3.2.1 Discrete Random Variable

Definition 3.2.1 — Discrete Random Variable

A random variable \(X\) is called discrete if it can take only a countable number of distinct values — either a finite set like \(\{ 0,1,2,3\}\) or a countably infinite set like \(\{ 0,1,2,3,\ldots\}\).

Simple test: Can you list all possible values of \(X\), even if the list is long? If yes — it is discrete.

Examples of Discrete Random Variables:

  • Number of heads in 5 coin tosses → \(X \in \{ 0,1,2,3,4,5\}\)
  • Number of students absent in a class of 60 → \(X \in \{ 0,1,2,\ldots,60\}\)
  • Number of calls received at a helpline per hour → \(X \in \{ 0,1,2,3,\ldots\}\)
  • Number of accidents on a road per day → \(X \in \{ 0,1,2,3,\ldots\}\)
  • Score on a single die → \(X \in \{ 1,2,3,4,5,6\}\)
  • Number of defective items in a batch of 100 → \(X \in \{ 0,1,2,\ldots,100\}\)

3.2.2 Continuous Random Variable

Definition 3.2.2 — Continuous Random Variable

A random variable \(X\) is called continuous if it can take any value in a given interval (or union of intervals). The set of possible values is uncountably infinite — you cannot list them out.

Simple test: Does \(X\) measure something on a continuous scale (height, weight, time, temperature, distance)? If yes — it is continuous.

Examples of Continuous Random Variables:

  • Height of a student (e.g., 165.3 cm, 172.84 cm, ...)
  • Weight of a newborn baby (e.g., 2.7 kg, 3.125 kg, ...)
  • Time taken to finish an exam (any value in, say, \(\lbrack 0,180\rbrack\) minutes)
  • Temperature of a city on a given day
  • Lifetime of an electric bulb (in hours)
  • Daily rainfall in mm
  • Speed of a vehicle on a highway

3.2.3 A Very Important Difference

For a discrete random variable, we can say \(P(X = 3) = 0.2\) — this is perfectly meaningful.

For a continuous random variable, \(P(X = x) = 0\) for every specific value \(x\). This might seem strange at first. Here is why:

If height \(X\) is continuous, the probability that a person is exactly 165.000000... cm is zero — because there are infinitely many possible exact values, and no single one carries any probability. Instead, we talk about probability over an interval:

\[P(164 \leq X \leq 166) = \text{some positive number}\]

3.3 Probability Function, PMF, and PDF

Now the key question: how do we describe or specify the probability behaviour of a random variable? The answer is through a probability function.

3.3.1 Probability Function — General Idea

A probability function is a mathematical rule (formula or table) that tells us the probability associated with each value (or range of values) of a random variable.

Think of it like a menu card — it lists what values \(X\) can take and what probability is attached to each.

There are two versions of this idea — one for discrete variables (called PMF) and one for continuous variables (called PDF).

3.3.2 Probability Mass Function (PMF) — For Discrete R.V.

Definition 3.3.1 — PMF

For a discrete random variable \(X\), the Probability Mass Function (PMF) is a function \(p(x)\) defined as:

\[p(x) = P(X = x)\quad\text{for each possible value }x\text{ of }X\]

Conditions for a valid PMF: A function \(p(x)\) is a valid PMF if and only if:

\[\text{(i)}\quad p(x_{i}) \geq 0\quad\text{for all }i\] \[\text{(ii)}\quad\sum_{\text{all }x_{i}}^{}p(x_{i}) = 1\]

The first condition says probabilities cannot be negative. The second says all probabilities must add up to 1.

Example 3.3.1 — PMF of Two Coin Tosses

Let \(X\) = number of heads in two tosses of a fair coin.

Sample space: \(\{ HH,HT,TH,TT\}\) — all outcomes equally likely with probability \(\frac{1}{4}\).

\(x\)Outcomes giving \(X = x\)\(p(x) = P(X = x)\)
0TT\(\frac{1}{4} = 0.25\)
1HT, TH\(\frac{2}{4} = 0.50\)
2HH\(\frac{1}{4} = 0.25\)

Verification:

\[p(0) + p(1) + p(2) = \frac{1}{4} + \frac{2}{4} + \frac{1}{4} = 1\]

This table completely describes the probability behaviour of \(X\). It is the PMF of \(X\).

Example 3.3.2 — PMF of a Fair Die

Let \(X\) = number shown on rolling a fair die.

\[p(x) = \frac{1}{6},\quad x = 1,2,3,4,5,6\]

All values are equally likely. Verification: \(6 \times \frac{1}{6} = 1\)

This is called a Discrete Uniform Distribution.

Example 3.3.3 — Finding the Unknown Constant \(k\)

A discrete random variable \(X\) has PMF:

\[p(x) = kx,\quad x = 1,2,3,4\quad\text{(and 0 elsewhere)}\]

Find the value of \(k\).

Solution:

Using the condition \(\sum p(x) = 1\):

\[p(1) + p(2) + p(3) + p(4) = 1\]
\[k(1) + k(2) + k(3) + k(4) = 1\]
\[k(1 + 2 + 3 + 4) = 1\]
\[10k = 1 \Longrightarrow k = \frac{1}{10}\]

The PMF is \(p(x) = \frac{x}{10}\) for \(x = 1,2,3,4\).

\(x\)1234
\(p(x)\)\(\frac{1}{10}\)\(\frac{2}{10}\)\(\frac{3}{10}\)\(\frac{4}{10}\)

Verification: \(\frac{1 + 2 + 3 + 4}{10} = \frac{10}{10} = 1\)

Example 3.3.4 — PMF of Number of Defective Bulbs

Two bulbs are drawn from a box containing 3 good and 2 defective bulbs (without replacement). Let \(X\) = number of defective bulbs drawn.

\(X\) can take values \(0,1,2\).

\[P(X = 0) = \frac{\binom{2}{0}\binom{3}{2}}{\binom{5}{2}} = \frac{1 \times 3}{10} = \frac{3}{10}\]
\[P(X = 1) = \frac{\binom{2}{1}\binom{3}{1}}{\binom{5}{2}} = \frac{2 \times 3}{10} = \frac{6}{10}\]
\[P(X = 2) = \frac{\binom{2}{2}\binom{3}{0}}{\binom{5}{2}} = \frac{1 \times 1}{10} = \frac{1}{10}\]

Verification: \(\frac{3 + 6 + 1}{10} = 1\)

3.3.3 Probability Density Function (PDF) — For Continuous R.V.

Since a continuous random variable takes uncountably many values, we cannot assign probability to individual points. Instead, we describe how densely the probability is spread using a Probability Density Function (PDF).

Definition 3.3.2 — PDF

A function \(f(x)\) is called the Probability Density Function (PDF) of a continuous random variable \(X\) if it satisfies:

\[\text{(i)}\quad f(x) \geq 0\quad\text{for all }x \in ( - \infty, + \infty)\] \[\text{(ii)}\quad\int_{- \infty}^{\infty}f(x)\, dx = 1\]

And the probability of \(X\) lying in the interval \(\lbrack a,b\rbrack\) is:

\[P(a \leq X \leq b) = \int_{a}^{b}f(x)\, dx\]

This is the area under the curve \(f(x)\) from \(a\) to \(b\).

Critical Point: \(f(x)\) is not the probability itself. It is the probability density — probability per unit length. Actual probability is the area under \(f(x)\), not its height. Also, \(f(x)\) can be greater than 1 at some points, and that is perfectly valid, as long as the total area equals 1.

Also Note: For a continuous random variable, since \(P(X = a) = \int_{a}^{a}f(x)\, dx = 0\), it does not matter whether we write \(<\), \(\leq\), \(>\), or \(\geq\) in probability:
\[P(a < X < b) = P(a \leq X \leq b) = P(a \leq X < b) = \int_{a}^{b}f(x)\, dx\]

Example 3.3.5 — Verifying a PDF

\[f(x) = 2x,\quad 0 \leq x \leq 1\quad\text{(and 0 elsewhere)}\]

Step 1: Check non-negativity. For \(0 \leq x \leq 1\): \(f(x) = 2x \geq 0\)

Step 2: Check total area = 1.

\[\int_{0}^{1}2x\, dx = \left\lbrack x^{2} \right\rbrack_{0}^{1} = 1 - 0 = 1\]

This is a valid PDF.

Step 3: Find \(P(0.25 \leq X \leq 0.75)\).

\[P(0.25 \leq X \leq 0.75) = \int_{0.25}^{0.75}2x\, dx = \left\lbrack x^{2} \right\rbrack_{0.25}^{0.75} = (0.75)^{2} - (0.25)^{2}\]
\[= 0.5625 - 0.0625 = 0.50\]

So there is a 50% probability that \(X\) lies in this interval.

Example 3.3.6 — Finding the Constant \(c\)

\[f(x) = c(1 - x^{2}),\quad 0 \leq x \leq 1\]

Find \(c\).

Using \(\int_{0}^{1}f(x)\, dx = 1\):

\[\int_{0}^{1}c(1 - x^{2})\, dx = 1\]
\[c\left\lbrack x - \frac{x^{3}}{3} \right\rbrack_{0}^{1} = 1\]
\[c\left( 1 - \frac{1}{3} \right) = 1\]
\[c \cdot \frac{2}{3} = 1 \Longrightarrow c = \frac{3}{2}\]

Example 3.3.7 — PDF with Two Sub-intervals

\[f(x) = \left\{ \begin{matrix} x & 0 \leq x \leq 1 \\ 2 - x & 1 < x \leq 2 \\ 0 & \text{elsewhere} \end{matrix} \right.\]

Verify this is a valid PDF.

\[\int_{0}^{1}x\, dx + \int_{1}^{2}(2 - x)\, dx = \left\lbrack \frac{x^{2}}{2} \right\rbrack_{0}^{1} + \left\lbrack 2x - \frac{x^{2}}{2} \right\rbrack_{1}^{2}\]
\[= \frac{1}{2} + \left\lbrack (4 - 2) - \left( 2 - \frac{1}{2} \right) \right\rbrack = \frac{1}{2} + \left\lbrack 2 - \frac{3}{2} \right\rbrack = \frac{1}{2} + \frac{1}{2} = 1\]

Find \(P(0.5 \leq X \leq 1.5)\).

\[P(0.5 \leq X \leq 1.5) = \int_{0.5}^{1}x\, dx + \int_{1}^{1.5}(2 - x)\, dx\]
\[= \left\lbrack \frac{x^{2}}{2} \right\rbrack_{0.5}^{1} + \left\lbrack 2x - \frac{x^{2}}{2} \right\rbrack_{1}^{1.5}\]
\[= \left( \frac{1}{2} - \frac{1}{8} \right) + \left\lbrack \left( 3 - \frac{9}{8} \right) - \left( 2 - \frac{1}{2} \right) \right\rbrack\]
\[= \frac{3}{8} + \left\lbrack \frac{15}{8} - \frac{3}{2} \right\rbrack = \frac{3}{8} + \frac{3}{8} = \frac{3}{4} = 0.75\]

3.4 Distribution Function and Its Properties

We have seen PMF and PDF as tools for describing a random variable. There is another extremely useful tool called the Cumulative Distribution Function (CDF), or simply the Distribution Function.

3.4.1 Definition of CDF

Definition 3.4.1 — CDF

The Cumulative Distribution Function (CDF) of a random variable \(X\) is defined as:

\[F(x) = P(X \leq x),\quad - \infty < x < + \infty\]

In plain words: \(F(x)\) tells us the probability that \(X\) takes a value less than or equal to \(x\).

Think of CDF as a running total. If PMF is like your daily pocket money, the CDF is like the total money in your piggy bank up to that day!

3.4.2 CDF for a Discrete Random Variable

For a discrete random variable:

\[F(x) = P(X \leq x) = \sum_{x_{i} \leq x}^{}p(x_{i})\]

That is, add up all PMF values for all \(x_{i}\) that are \(\leq x\).

Example 3.4.1 — CDF from the Two-Coin PMF

From Example 3.3.1: \(p(0) = \frac{1}{4}\), \(p(1) = \frac{2}{4}\), \(p(2) = \frac{1}{4}\)

Range of \(x\)\(F(x) = P(X \leq x)\)Calculation
\(x < 0\)\(0\)No values below 0
\(0 \leq x < 1\)\(\frac{1}{4} = 0.25\)\(p(0)\)
\(1 \leq x < 2\)\(\frac{3}{4} = 0.75\)\(p(0) + p(1) = \frac{1}{4} + \frac{2}{4}\)
\(x \geq 2\)\(1.00\)\(p(0) + p(1) + p(2)\)

Notice: \(F(x)\) starts at 0, jumps at each value of \(X\), and finally reaches 1. For discrete variables, the CDF is a step function.

Example 3.4.2 — CDF of Die Roll

\(X\) = outcome on a fair die; \(p(x) = \frac{1}{6}\) for \(x = 1,2,...,6\).

\[F(x) = \frac{\lfloor x\rfloor}{6}\quad\text{for }1 \leq x \leq 6\]

For example: \(F(3.5) = P(X \leq 3.5) = P(X \leq 3) = \frac{3}{6} = \frac{1}{2}\).

3.4.3 CDF for a Continuous Random Variable

For a continuous random variable:

\[F(x) = P(X \leq x) = \int_{- \infty}^{x}f(t)\, dt\]

And conversely, differentiating both sides:

\[f(x) = \frac{d}{dx}F(x)\]

The PDF is the derivative of the CDF.

Example 3.4.3 — CDF from a Given PDF

Given \(f(x) = 2x\) for \(0 \leq x \leq 1\). Find \(F(x)\).

Case 1: \(x < 0\)

\[F(x) = 0\]

Case 2: \(0 \leq x \leq 1\)

\[F(x) = \int_{0}^{x}2t\, dt = \left\lbrack t^{2} \right\rbrack_{0}^{x} = x^{2}\]

Case 3: \(x > 1\)

\[F(x) = 1\]

Therefore:

\[F(x) = \left\{ \begin{matrix} 0 & x < 0 \\ x^{2} & 0 \leq x \leq 1 \\ 1 & x > 1 \end{matrix} \right.\]

Check: \(F(0.5) = (0.5)^{2} = 0.25\), meaning \(P(X \leq 0.5) = 25\%\). Makes sense since the PDF is higher near \(x = 1\) (triangular-shaped).

Example 3.4.4 — CDF of Exponential Type

\[f(x) = e^{- x},\quad x > 0\]
\[F(x) = \int_{0}^{x}e^{- t}\, dt = \left\lbrack - e^{- t} \right\rbrack_{0}^{x} = 1 - e^{- x},\quad x > 0\]

Find \(P(1 < X < 2)\):

\[P(1 < X < 2) = F(2) - F(1) = (1 - e^{- 2}) - (1 - e^{- 1}) = e^{- 1} - e^{- 2}\]
\[\approx 0.3679 - 0.1353 = 0.2326\]

3.4.4 Properties of the Distribution Function

These are very important for examinations. Learn them carefully.

Property 1: \(F(x)\) is non-decreasing.

If \(x_{1} < x_{2}\), then \(F(x_{1}) \leq F(x_{2})\).

Reason: If \(x\) increases, more values are included in \(P(X \leq x)\), so it can only stay same or increase.


Property 2: \(F( - \infty) = 0\)

\[\lim_{x \rightarrow - \infty}F(x) = 0\]

Reason: \(P(X \leq - \infty) = 0\) — nothing can be less than \(- \infty\).


Property 3: \(F( + \infty) = 1\)

\[\lim_{x \rightarrow + \infty}F(x) = 1\]

Reason: \(P(X \leq + \infty) = 1\) — \(X\) must take some finite value.


Property 4: \(F(x)\) is right-continuous.

\[F(x^{+}) = F(x)\quad\text{for all }x\]

Property 5: Interval probability:

\[P(a < X \leq b) = F(b) - F(a)\]

This is the most frequently used property in calculations.


Property 6: For any \(x\):

\[P(X > x) = 1 - F(x)\]

Property 7: \(0 \leq F(x) \leq 1\) for all \(x\).


Property 8 (Discrete): \(F(x)\) is a step function — it is constant between values and jumps at each possible value of \(X\).


Property 9 (Continuous): \(F(x)\) is a continuous and smooth function everywhere.

Example 3.4.5 — Using CDF Properties

Given \(F(x) = x^{2}\) for \(0 \leq x \leq 1\). Find:

(a) \(P(0.3 < X \leq 0.7)\)

\[P(0.3 < X \leq 0.7) = F(0.7) - F(0.3) = (0.7)^{2} - (0.3)^{2} = 0.49 - 0.09 = 0.40\]

(b) \(P(X > 0.6)\)

\[P(X > 0.6) = 1 - F(0.6) = 1 - (0.6)^{2} = 1 - 0.36 = 0.64\]

(c) \(P(X \leq 0.4)\)

\[P(X \leq 0.4) = F(0.4) = (0.4)^{2} = 0.16\]

Example 3.4.6 — Finding PDF from CDF

Given:

\[F(x) = \left\{ \begin{matrix} 0 & x < 0 \\ \frac{x^{3}}{8} & 0 \leq x \leq 2 \\ 1 & x > 2 \end{matrix} \right.\]

Find the PDF \(f(x)\).

Differentiating \(F(x)\):

\[f(x) = \frac{d}{dx}F(x) = \frac{d}{dx}\left( \frac{x^{3}}{8} \right) = \frac{3x^{2}}{8},\quad 0 \leq x \leq 2\]

Verify: \(\int_{0}^{2}\frac{3x^{2}}{8}\, dx = \frac{3}{8}\left\lbrack \frac{x^{3}}{3} \right\rbrack_{0}^{2} = \frac{3}{8} \cdot \frac{8}{3} = 1\)

3.5 Functions of Random Variables

Very often in practice, we are not interested in \(X\) itself, but in some function of \(X\). For example:

  • If \(X\) = radius of a circle, we may want \(Y = \pi X^{2}\) (area).
  • If \(X\) = temperature in Celsius, we may want \(Y = \frac{9}{5}X + 32\) (Fahrenheit).
  • If \(X\) = a measurement, we may want \(Y = X^{2}\) or \(Y = \ln X\) or \(Y = |X|\).

Since \(X\) is a random variable and \(Y = g(X)\), \(Y\) is also a random variable. The question is: what is the distribution of \(Y\)?

3.5.1 Functions of Discrete Random Variables

This is the simpler case. If \(X\) has PMF \(p(x)\) and \(Y = g(X)\), then:

\[P(Y = y) = \sum_{\{ x\,:\, g(x) = y\}}^{}p(x)\]

In words: add up \(p(x)\) for all \(x\) values that give \(g(x) = y\).

Example 3.5.1 — \(Y = X^{2}\) for a Die

\(X\) = outcome on a fair die; \(p(x) = \frac{1}{6}\) for \(x = 1,2,...,6\).

Define \(Y = X^{2}\).

\(Y = X^{2}\) takes values \(\{ 1,4,9,16,25,36\}\), each with probability \(\frac{1}{6}\).

\(X\) takes values \(- 1,0,1\) with probabilities \(0.3,0.4,0.3\).

Example 3.5.3 — Two Values of \(X\) Map to Same \(Y\)

Let \(X\) take values \(- 2, - 1,0,1,2\) each with probability \(\frac{1}{5}\).

3.5.2 Functions of Continuous Random Variables — CDF Method

For continuous random variables, finding the distribution of \(Y = g(X)\) uses the following procedure:

Step 1: Find the CDF of \(Y\):

\[F_{Y}(y) = P(Y \leq y) = P(g(X) \leq y)\]

Express this in terms of \(X\) and evaluate using the CDF of \(X\).

Step 2: Differentiate to get the PDF of \(Y\):

\[f_{Y}(y) = \frac{d}{dy}F_{Y}(y)\]

Example 3.5.4 — \(Y = X^{2}\) when \(X\) has \(f(x) = 2x\), \(0 \leq x \leq 1\)

We know \(F_{X}(x) = x^{2}\) (from Example 3.4.3).

Find the distribution of \(Y = X^{2}\).

Step 1: Find \(F_{Y}(y)\).

For \(0 \leq y \leq 1\):

\[F_{Y}(y) = P(Y \leq y) = P(X^{2} \leq y) = P(X \leq \sqrt{y})\]

(Since \(X \geq 0\), \(X^{2} \leq y \Leftrightarrow X \leq \sqrt{y}\).)

\[F_{Y}(y) = F_{X}(\sqrt{y}) = (\sqrt{y})^{2} = y\]

Step 2: Differentiate:

\[f_{Y}(y) = \frac{d}{dy}(y) = 1,\quad 0 \leq y \leq 1\]

So \(Y \sim \text{Uniform}\lbrack 0,1\rbrack\) — an interesting result!

Example 3.5.5 — Linear Transformation \(Y = aX + b\)

Let \(X\) have PDF \(f_{X}(x)\) and CDF \(F_{X}(x)\). Define \(Y = aX + b\) where \(a > 0\).

\[F_{Y}(y) = P(Y \leq y) = P(aX + b \leq y) = P\left( X \leq \frac{y - b}{a} \right) = F_{X}\left( \frac{y - b}{a} \right)\]

Differentiating:

\[f_{Y}(y) = \frac{1}{a}f_{X}\left( \frac{y - b}{a} \right)\]

This is the general formula for a linear transformation of a continuous random variable.

3.6 Joint, Marginal, and Conditional Probability Distributions

So far we have studied one random variable at a time. But in real life, we often have two (or more) random variables observed together from the same experiment. For example:

  • Marks in Physics (\(X\)) and marks in Chemistry (\(Y\)) of the same student.
  • Height (\(X\)) and weight (\(Y\)) of the same person.
  • Daily temperature (\(X\)) and humidity (\(Y\)) of the same city.

To understand such situations, we study bivariate distributions — distributions of two random variables together.

3.6.1 Joint Probability Distribution — Discrete Case

Definition 3.6.1 — Joint PMF

The Joint Probability Mass Function (Joint PMF) of two discrete random variables \(X\) and \(Y\) is:

\[p(x,y) = P(X = x,\ Y = y)\]

for all pairs \((x,y)\) that \(X\) and \(Y\) can jointly take.

Conditions for a valid Joint PMF:

\[\text{(i)}\quad p(x,y) \geq 0\quad\text{for all }(x,y)\] \[\text{(ii)}\quad\sum_{x}^{}{\sum_{y}^{}p}(x,y) = 1\]

Example 3.6.1 — Two Fair Dice

Two fair dice are rolled. Let \(X\) = score on first die, \(Y\) = score on second die.

Since the dice are independent:

\[p(x,y) = \frac{1}{6} \cdot \frac{1}{6} = \frac{1}{36},\quad x = 1,...,6;\ y = 1,...,6\]

All 36 pairs are equally likely.

Find \(P(X + Y = 5)\):

Pairs summing to 5: \((1,4),(2,3),(3,2),(4,1)\) — four pairs.

\[P(X + Y = 5) = 4 \times \frac{1}{36} = \frac{1}{9}\]

Example 3.6.2 — Joint PMF in a Table

A bag has 2 red (R) and 3 black (B) balls. Two balls are drawn without replacement. Define:

\(X\) = number of red balls among the two drawn. \(Y\) = number of black balls among the two drawn.

Note: \(X + Y = 2\) always, so \(Y = 2 - X\), but we can still write the joint table.

\(Y = 0\)\(Y = 1\)\(Y = 2\)Row Total
\(X = 0\)00\(\frac{3}{10}\)\(\frac{3}{10}\)
\(X = 1\)0\(\frac{6}{10}\)0\(\frac{6}{10}\)
\(X = 2\)\(\frac{1}{10}\)00\(\frac{1}{10}\)
Col Total\(\frac{1}{10}\)\(\frac{6}{10}\)\(\frac{3}{10}\)1

Verification: \(\frac{3}{10} + \frac{6}{10} + \frac{1}{10} = 1\)

Example 3.6.3 — Constructing a Joint PMF

The joint PMF of \((X,Y)\) is given as:

\[p(x,y) = \frac{x + y}{21},\quad x = 1,2,3;\ y = 1,2\]

Verify it is a valid joint PMF.

\(y = 1\)\(y = 2\)
\(x = 1\)\(\frac{2}{21}\)\(\frac{3}{21}\)
\(x = 2\)\(\frac{3}{21}\)\(\frac{4}{21}\)
\(x = 3\)\(\frac{4}{21}\)\(\frac{5}{21}\)
\[\text{Sum} = \frac{2 + 3 + 3 + 4 + 4 + 5}{21} = \frac{21}{21} = 1\]

3.6.2 Joint PDF — Continuous Case

Definition 3.6.2 — Joint PDF

A function \(f(x,y)\) is the Joint Probability Density Function of two continuous random variables \(X\) and \(Y\) if:

\[\text{(i)}\quad f(x,y) \geq 0\quad\text{for all }(x,y)\] \[\text{(ii)}\quad\int_{- \infty}^{\infty}{\int_{- \infty}^{\infty}f}(x,y)\, dx\, dy = 1\]

The probability over a region \(A\) is:

\[P\left( (X,Y) \in A \right) = \iint_{A}^{}f(x,y)\, dx\, dy\]

3.6.3 Marginal Probability Distribution

From the joint distribution, we can recover the individual distributions of \(X\) and \(Y\) separately. These are called Marginal Distributions because they are found at the margins (totals) of the joint probability table.

For Discrete Random Variables:

\[p_{X}(x) = \sum_{y}^{}p(x,y)\quad\quad\text{(Marginal PMF of }X\text{)}\] \[p_{Y}(y) = \sum_{x}^{}p(x,y)\quad\quad\text{(Marginal PMF of }Y\text{)}\]

For Continuous Random Variables:

\[f_{X}(x) = \int_{- \infty}^{\infty}f(x,y)\, dy\quad\quad\text{(Marginal PDF of }X\text{)}\] \[f_{Y}(y) = \int_{- \infty}^{\infty}f(x,y)\, dx\quad\quad\text{(Marginal PDF of }Y\text{)}\]

Example 3.6.4 — Marginal PMF (from Example 3.6.3)

\[p(x,y) = \frac{x + y}{21},\quad x = 1,2,3;\ y = 1,2\]

Marginal PMF of \(X\):

\[p_{X}(1) = \frac{2}{21} + \frac{3}{21} = \frac{5}{21}\]
\[p_{X}(2) = \frac{3}{21} + \frac{4}{21} = \frac{7}{21}\]
\[p_{X}(3) = \frac{4}{21} + \frac{5}{21} = \frac{9}{21}\]

Verification: \(\frac{5 + 7 + 9}{21} = \frac{21}{21} = 1\)

Marginal PMF of \(Y\):

\[p_{Y}(1) = \frac{2}{21} + \frac{3}{21} + \frac{4}{21} = \frac{9}{21}\]
\[p_{Y}(2) = \frac{3}{21} + \frac{4}{21} + \frac{5}{21} = \frac{12}{21}\]

Verification: \(\frac{9 + 12}{21} = 1\)

Example 3.6.5 — Marginal PDFs (Continuous)

Given joint PDF:

\[f(x,y) = 2,\quad 0 \leq x \leq y \leq 1\]

Find the marginal PDFs of \(X\) and \(Y\).

Marginal PDF of \(X\):

For a fixed \(x\), \(y\) ranges from \(x\) to \(1\):

\[f_{X}(x) = \int_{x}^{1}2\, dy = 2(1 - x),\quad 0 \leq x \leq 1\]

Marginal PDF of \(Y\):

For a fixed \(y\), \(x\) ranges from \(0\) to \(y\):

\[f_{Y}(y) = \int_{0}^{y}2\, dx = 2y,\quad 0 \leq y \leq 1\]

Verification:

\[\int_{0}^{1}2(1 - x)\, dx = 2\left\lbrack x - \frac{x^{2}}{2} \right\rbrack_{0}^{1} = 2\left( 1 - \frac{1}{2} \right) = 1\]
\[\int_{0}^{1}2y\, dy = \left\lbrack y^{2} \right\rbrack_{0}^{1} = 1\]

Example 3.6.6 — Marginal PDFs with Exponential Type

\[f(x,y) = e^{- (x + y)},\quad x > 0,\ y > 0\]

Marginal PDF of \(X\):

\[f_{X}(x) = \int_{0}^{\infty}e^{- (x + y)}\, dy = e^{- x}\int_{0}^{\infty}e^{- y}\, dy = e^{- x} \cdot 1 = e^{- x},\quad x > 0\]

Marginal PDF of \(Y\):

By symmetry: \(f_{Y}(y) = e^{- y}\), \(y > 0\).

Note: \(f(x,y) = e^{- x} \cdot e^{- y} = f_{X}(x) \cdot f_{Y}(y)\), confirming \(X\) and \(Y\) are independent.

3.6.4 Conditional Probability Distribution

The conditional distribution of \(X\) given \(Y = y\) answers the question:

"If we already know that \(Y = y\), what is the distribution of \(X\)?"

It uses the extra information about \(Y\) to refine our knowledge about \(X\).

For Discrete Random Variables:

\[P(X = x \mid Y = y) = \frac{p(x,y)}{p_{Y}(y)},\quad\text{provided }p_{Y}(y) > 0\]

For Continuous Random Variables:

\[f(x \mid y) = \frac{f(x,y)}{f_{Y}(y)},\quad\text{provided }f_{Y}(y) > 0\] \[f(y \mid x) = \frac{f(x,y)}{f_{X}(x)},\quad\text{provided }f_{X}(x) > 0\]

Example 3.6.7 — Conditional PMF (Discrete)

From Example 3.6.3: \(p(x,y) = \frac{x + y}{21}\); \(p_{Y}(1) = \frac{9}{21}\).

Find the conditional distribution of \(X\) given \(Y = 1\).

\[P(X = 1 \mid Y = 1) = \frac{p(1,1)}{p_{Y}(1)} = \frac{2/21}{9/21} = \frac{2}{9}\]
\[P(X = 2 \mid Y = 1) = \frac{p(2,1)}{p_{Y}(1)} = \frac{3/21}{9/21} = \frac{3}{9} = \frac{1}{3}\]
\[P(X = 3 \mid Y = 1) = \frac{p(3,1)}{p_{Y}(1)} = \frac{4/21}{9/21} = \frac{4}{9}\]

Verification: \(\frac{2}{9} + \frac{3}{9} + \frac{4}{9} = \frac{9}{9} = 1\)

Example 3.6.8 — Conditional PDF (Continuous)

From Example 3.6.5: \(f(x,y) = 2\) for \(0 \leq x \leq y \leq 1\), and \(f_{Y}(y) = 2y\).

Find \(f(x \mid y)\).

\[f(x \mid y) = \frac{f(x,y)}{f_{Y}(y)} = \frac{2}{2y} = \frac{1}{y},\quad 0 \leq x \leq y\]

This is a Uniform distribution on \(\lbrack 0,y\rbrack\).

Interpretation: Given that \(Y = y\), the random variable \(X\) is uniformly spread between 0 and \(y\).

Find \(P\left( X < \frac{y}{2}\,|\, Y = y \right)\):

\[P\left( X < \frac{y}{2}\,|\, Y = y \right) = \int_{0}^{y/2}\frac{1}{y}\, dx = \frac{1}{y} \cdot \frac{y}{2} = \frac{1}{2}\]

3.6.5 Independence of Random Variables

Definition 3.6.3 — Independence

Two random variables \(X\) and \(Y\) are said to be independent if their joint distribution equals the product of their marginal distributions:

Discrete case:

\[p(x,y) = p_{X}(x) \cdot p_{Y}(y)\quad\text{for all }(x,y)\]

Continuous case:

\[f(x,y) = f_{X}(x) \cdot f_{Y}(y)\quad\text{for all }(x,y)\]

Intuition: \(X\) and \(Y\) are independent when knowing the value of one gives you no information about the other.

Example 3.6.9 — Checking Independence (Discrete)

Joint PMF: \(p(x,y) = \frac{x + y}{21}\) for \(x = 1,2,3\); \(y = 1,2\).

From Example 3.6.4: \(p_{X}(1) = \frac{5}{21}\), \(p_{Y}(1) = \frac{9}{21}\).

Check: \(p_{X}(1) \cdot p_{Y}(1) = \frac{5}{21} \times \frac{9}{21} = \frac{45}{441}\)

But \(p(1,1) = \frac{2}{21} = \frac{42}{441}\)

Since \(\frac{45}{441} eq \frac{42}{441}\), \(X\) and \(Y\) are NOT independent.

Example 3.6.10 — Checking Independence (Continuous)

From Example 3.6.6: \(f(x,y) = e^{- (x + y)}\), \(f_{X}(x) = e^{- x}\), \(f_{Y}(y) = e^{- y}\).

\[f_{X}(x) \cdot f_{Y}(y) = e^{- x} \cdot e^{- y} = e^{- (x + y)} = f(x,y)\]

\(X\) and \(Y\) are independent.

Q1. Define a random variable. Give two examples from daily life.

A random variable is a real-valued function defined on the sample space of a random experiment. Examples: 1. Number of heads in 3 coin tosses. 2. Lifespan of a bulb in hours.

(i) \(p(x) \geq 0\) for all \(x\)
(ii) \(\sum p(x) = 1\)

(i) \(f(x) \geq 0\) for all \(x\)
(ii) \(\int_{-\infty}^{\infty} f(x) dx = 1\)

Q5. Define CDF. Write the formula for finding \(P(a < X \leq b)\) using CDF.

The CDF \(F(x)\) is defined as \(F(x) = P(X \leq x)\).
\(P(a < X \leq b) = F(b) - F(a)\).

Q6. List any five properties of the distribution function \(F(x)\).

1. Non-decreasing: If \(x_1 < x_2\), \(F(x_1) \leq F(x_2)\).
2. \(F(-\infty) = 0\)
3. \(F(+\infty) = 1\)
4. Right-continuous: \(F(x^+) = F(x)\)
5. \(0 \leq F(x) \leq 1\)

Q7. What is the relationship between PDF \(f(x)\) and CDF \(F(x)\) for a continuous random variable?

\(f(x) = \frac{d}{dx} F(x)\) and \(F(x) = \int_{-\infty}^{x} f(t) dt\).

Q8. Define marginal distribution. How is it obtained from the joint distribution?

Marginal distribution gives the individual distribution of one variable from a joint distribution.
For discrete: \(p_X(x) = \sum_y p(x,y)\).
For continuous: \(f_X(x) = \int_{-\infty}^{\infty} f(x,y) dy\).

Q9. What is a conditional distribution? Give its formula for both discrete and continuous cases.

It's the distribution of one variable given the value of another.
Discrete: \(P(X=x | Y=y) = \frac{p(x,y)}{p_Y(y)}\)
Continuous: \(f(x|y) = \frac{f(x,y)}{f_Y(y)}\)

Q10. When are two random variables \(X\) and \(Y\) said to be independent?

When their joint distribution equals the product of their marginal distributions.
Discrete: \(p(x,y) = p_X(x) p_Y(y)\)
Continuous: \(f(x,y) = f_X(x) f_Y(y)\)

\(Y\) takes values \(2,4,6,8,10,12\).

\(f_Y(y) = F_Y'(y) = 1\) for \(0 < y < 1\). Thus \(Y \sim \text{Uniform}(0,1)\).
