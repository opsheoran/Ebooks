Systematic Random Sampling

A sampling technique in which only the first unit is selected with the help of random numbers and the rest get selected automatically according to some pre-designed pattern is known as systematic random sampling. Suppose N units of the population are numbered from 1 to N in some order. Let N=nk, where n is the sample size and k is an integer, called sampling interval.

Select a random number less than or equal to k be selected and every \(k^{th}\) unit thereafter. The resultant sample is called every \(k^{th}\) systematic sample and such a procedure termed linear systematic sampling.

Theorem 5.1: If N=nk the systematic sample mean \(\bar{y}_{sys}\) is an unbiased estimate of the population mean \(\bar{Y}\).

Circular Systematic Sampling

If N is not a multiple of n, the linear systematic sampling does not provide a fixed sample size n. In such cases, Circular Systematic Sampling is used.

  1. Select a random number \(r\) from 1 to N.
  2. The units with serial numbers \((r + j k) \pmod N\) for \(j = 0, 1, \ldots, n-1\) are selected.

This ensures that each unit in the population has an equal chance of being included in the sample.

Merits and Demerits

Merits:

  • Operationally more convenient than SRS or Stratified.
  • Time and work involved is relatively much less.
  • Yields a sample which is evenly spread over the entire population.

Demerits:

  • If N is not a multiple of n, the actual sample size is different from that requirement.
  • Sample mean is not an unbiased estimate if N is not a multiple of n in linear systematic sampling.
  • May yield highly biased estimates if there are periodic features associated with the sampling interval.
